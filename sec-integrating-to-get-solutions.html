    Section 1.2.1 Integrating to solve \(\frac{dy}{dx} = f(x)\)

    A simple but important special case of a first-order ODE is when the right-hand side does not involve the dependent variable. That is, the equation is in the form:
    \begin{equation*} \frac{dy}{dx} = f(x). \end{equation*}
    🔗
    Let’s explore this with the following example.
    🔗

    Example 1.2.1. A Simple First-Order ODE.

    Suppose \(\frac{dy}{dx} = 2x\text{.}\) What does \(y(x)\) equal?
    🔗
    Hint.
    Think about what operation undoes differentiation. What function has derivative equal to \(2x\text{?}\)
    🔗
    🔗
    Solution.
    Notice that \(y(x) = x^2\) is a solution since its derivative is \(2x\text{.}\) That is, we found an antiderivative of \(2x\text{.}\)
    🔗
    Recall that adding any constant \(C\) gives the general antiderivative \(y(x) = x^2 + C\text{,}\) which is the general solution to \(\frac{dy}{dx} = 2x\text{.}\) So we simply took the indefinite integral of \(2x\text{!}\)
    🔗
    🔗
    🔗
    The graph with an interactive slider below demonstrates what is happening in this example.
    🔗
    Alright, well that turned out to be pretty easy once we understood what the differential equation was saying! We generalize this idea in the following observation.
    🔗

    Observation 1.2.3.

    For a differential equation in the form
    \begin{equation*} \frac{dy}{dx} = f(x), \end{equation*}
    the general solution is simply:
    \begin{equation*} y(x) = \int f(x)\, dx + C. \end{equation*}
    🔗
    🔗
    Let’s use this principle in the following examples. Try them on your own first!
    🔗

    Activity 1.2.1. Practice: Integrating to Find Solutions.

    Try each of the following on your own before checking the solution. Give it a shot first!
    🔗

    (a)

    Solve \(\dfrac{dy}{dx} = 3x^2\) with initial condition \(y(1) = 3\text{.}\)
    🔗
    Solution.
    Integrating both sides:
    \begin{equation*} y = \int 3x^2\, dx = x^3 + C. \end{equation*}
    Applying the initial condition \(y(1) = 3\text{:}\)
    \begin{equation*} 3 = 1^3 + C \implies C = 2. \end{equation*}
    So the specific solution is:
    \begin{equation*} y(x) = x^3 + 2. \end{equation*}
    We can verify: \(\frac{dy}{dx} = 3x^2\) \(\checkmark\) and \(y(1) = 1 + 2 = 3\) \(\checkmark\text{.}\)
    🔗
    🔗
    🔗

    (b)

    Find the general solution to \(\dfrac{d^2p}{dt^2} = 3\text{.}\)
    🔗
    Hint.
    Notice the left-hand side involves a second derivative, which we can write as \(\frac{d}{dt}\!\left(\frac{dp}{dt}\right) = 3\text{.}\) Apply the same integration principle to get \(\frac{dp}{dt} = \int 3\, dt\text{,}\) then integrate again.
    🔗
    🔗
    Solution.
    Since \(\frac{d^2p}{dt^2} = \frac{d}{dt}\!\left(\frac{dp}{dt}\right)\text{,}\) we integrate twice.
    🔗
    First integration: Starting from \(p'(t) = \int 3\, dt\text{:}\)
    🔗
    Solution 1. Reveal
    \begin{equation*} p'(t) = 3t + C_1. \end{equation*}
    🔗
    🔗
    Now that we have \(p'(t) = 3t + C_1\text{,}\) we integrate again.
    🔗
    Second integration: Starting from \(p(t) = \int (3t + C_1)\, dt\text{:}\)
    🔗
    Solution 2. Reveal
    \begin{equation*} p(t) = \frac{3}{2}t^2 + C_1 t + C_2. \end{equation*}
    🔗
    🔗
    You can verify that the general solution satisfies \(p''(t) = 3\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Note 1.2.4.

    Notice in the last example we obtained two constants of integration \(C_1\) and \(C_2\) from integrating twice.
    🔗
    🔗
    We now have a solid understanding of solving ODEs in the form \(\frac{dy}{dx} = f(x)\text{.}\) Does this mean we’ve solved the entire course? Not quite, ODEs are rarely this simple! The key feature of \(\frac{dy}{dx} = f(x)\) is that the right-hand side depends only on the independent variable. In general, we won’t be so lucky! Try the following checkpoint to determine when this method applies.
    🔗

    Checkpoint 1.2.5.

    Which of the following ODEs have been solved correctly using the integration method \(y = \int f(x)\, dx + C\text{?}\) Select all that apply.
    🔗
    • \(\frac{dy}{dt} = e^t\text{,}\) solved as \(y = \int e^t\, dt = e^t + C\text{.}\)
    • Correct! The right-hand side depends only on \(t\text{,}\) so we can integrate directly. \(\int e^t\, dt = e^t + C\text{.}\)
    • \(\frac{dy}{dx} = x^2 + 1\text{,}\) solved as \(y = \int (x^2 + 1)\, dx = \frac{x^3}{3} + x + C\text{.}\)
    • Correct! The right-hand side depends only on \(x\text{,}\) so we integrate directly to get \(\frac{x^3}{3} + x + C\text{.}\)
    • \(\frac{dy}{dx} = 2y\text{,}\) solved as \(y = \int 2y\, dx = y^2 + C\text{.}\)
    • Incorrect! The right-hand side depends on \(y\text{,}\) not just \(x\text{,}\) so we cannot simply integrate both sides this way. We will learn how to solve ODEs like this later in the course!
    • \(\frac{dP}{dt} = tP\text{,}\) solved as \(P = \int tP\, dt = \frac{t^2}{2} P + C\text{.}\)
    • Incorrect! The right-hand side depends on both \(t\) and \(P\text{,}\) so direct integration doesn’t work here. We will learn how to solve ODEs like this later in the course!
