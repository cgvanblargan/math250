    Section 1.4.5 Separation of Variables: Solving Explicitly Tricks

    The following example is a little harder to solve for \(y\) explicitly, but it’s certainly doable! It’s designed to give you practice executing some things that can come up during separation of variables, so try each step on your own before revealing the solution. And as always, the video at the end is there if you get stuck!
    🔗

    Example 1.4.1. Solving a Separable ODE Explicitly.

    Solve the following ODE explicitly: \(\frac{dy}{dx} = \frac{xe^{x^2}}{2y+4}\text{.}\)
    🔗

    (a) Step 1: Find equilibrium solutions.

    There are no equilibrium solutions.
    🔗
    Solution. Why?
    Rewrite the ODE as \(\frac{dy}{dx} = \frac{1}{2y+4} \cdot xe^{x^2}\text{,}\) so \(f(y) = \frac{1}{2y+4}\text{.}\) Setting \(f(y) = 0\) gives \(\frac{1}{2y+4} = 0\text{,}\) which has no solution. So there are no equilibrium solutions.
    🔗
    🔗
    🔗

    (b) Step 2: Separate the variables.

    Multiplying both sides by \((2y+4)\) and separating:
    \begin{equation*} \int (2y + 4)\, dy = \int xe^{x^2}\, dx. \end{equation*}
    🔗
    🔗

    (c) Step 3: Integrate both sides.

    Integrating the left-hand side:
    \begin{equation*} y^2 + 4y = \int xe^{x^2}\, dx. \end{equation*}
    What technique should you use to evaluate \(\int xe^{x^2}\, dx\text{?}\)
    🔗
    Hint.
    This integral can be solved with a \(u\)-substitution. What do you think you should set \(u\) equal to?
    🔗
    🔗
    Solution. Work: evaluating the integral
    Let \(u = x^2\text{,}\) so \(du = 2x\, dx\) and \(x\, dx = \frac{1}{2}du\text{:}\)
    \begin{align*} \int xe^{x^2}\, dx \amp= \int e^u \cdot \frac{1}{2}\, du\\ \amp= \frac{1}{2}e^u + C\\ \amp= \frac{1}{2}e^{x^2} + C. \end{align*}
    So we now have:
    \begin{equation*} y^2 + 4y = \frac{1}{2}e^{x^2} + C. \end{equation*}
    🔗
    🔗
    🔗

    (d) Step 4: Solve for \(y\) — recognizing the technique.

    We have \(y^2 + 4y = \frac{1}{2}e^{x^2} + C\text{.}\) What technique lets us solve explicitly for \(y\text{?}\)
    🔗
    • Complete the square on the left-hand side.
    • Exactly! Adding 4 to both sides gives \((y+2)^2 = \frac{1}{2}e^{x^2} + C\text{,}\) after which we can take a square root.
    • Divide both sides by \(y\text{.}\)
    • Dividing by \(y\) gives \(y + 4 = \frac{1}{y}\left(\frac{1}{2}e^{x^2} + C\right)\text{,}\) which doesn’t isolate \(y\text{.}\)
    • Take the square root of both sides directly.
    • Not quite — the left-hand side \(y^2 + 4y\) is not a perfect square yet, so we can’t take the square root directly. We need one more step first!
    • This cannot be solved explicitly for \(y\text{.}\)
    • It can be solved explicitly! Think about what algebraic technique turns \(y^2 + 4y\) into a perfect square.
    Solution. Working: completing the square
    To complete the square on \(y^2 + 4y\text{,}\) we add \(4\) to both sides (since \(\left(\frac{4}{2}\right)^2 = 4\)):
    \begin{align*} y^2 + 4y + 4 \amp= \frac{1}{2}e^{x^2} + C + 4\\ (y+2)^2 \amp= \frac{1}{2}e^{x^2} + C \end{align*}
    where we absorbed the \(+4\) into \(C\) on the right-hand side.
    🔗
    🔗
    🔗

    (e) Step 5: Write the explicit solution.

    Starting from \((y+2)^2 = \frac{1}{2}e^{x^2} + C\text{,}\) which of the following is the correct explicit solution?
    🔗
    • \(y = \pm\sqrt{\frac{1}{2}e^{x^2} + C} - 2\)
    • Correct! Taking the square root of both sides gives \(y + 2 = \pm\sqrt{\frac{1}{2}e^{x^2} + C}\text{,}\) and subtracting 2 gives the answer.
    • \(y = \sqrt{\frac{1}{2}e^{x^2} + C} - 2\)
    • Almost — but don’t forget the \(\pm\text{!}\) When taking the square root of both sides of \((y+2)^2 = \ldots\text{,}\) we get \(y+2 = \pm\sqrt{\ldots}\text{.}\)
    • \(y = \pm\sqrt{\frac{1}{2}e^{x^2} + C} + 2\)
    • Check the sign! From \(y + 2 = \pm\sqrt{\ldots}\text{,}\) subtracting 2 from both sides gives \(y = \pm\sqrt{\ldots} - 2\text{,}\) not \(+2\text{.}\)
    • \(y = \pm\sqrt{\frac{1}{2}e^{x^2}} + C - 2\)
    • Not quite — \(C\) should be inside the square root, not outside. We are taking the square root of the entire right-hand side \(\frac{1}{2}e^{x^2} + C\text{.}\)
    Solution. Working: taking the square root
    Taking the square root of both sides of \((y+2)^2 = \frac{1}{2}e^{x^2} + C\text{:}\)
    \begin{equation*} y + 2 = \pm\sqrt{\frac{1}{2}e^{x^2} + C}. \end{equation*}
    Subtracting 2 from both sides:
    \begin{equation*} y = \pm\sqrt{\frac{1}{2}e^{x^2} + C} - 2. \end{equation*}
    🔗
