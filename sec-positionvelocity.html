    Section 1.2.2 Velocity and Acceleration

    Let’s review some ideas regarding position, velocity, and acceleration.
    🔗

    Definition 1.2.1. Position, Velocity, and Acceleration.

    Suppose the position of an object with respect to time is given by \(p(t)\text{.}\)
    • The velocity \(v(t)\) is the rate of change of position: \(v(t) = p'(t)\text{.}\)
      🔗
    • The acceleration \(a(t)\) is the rate of change of velocity: \(a(t) = v'(t) = p''(t)\text{.}\)
      🔗
    🔗
    🔗
    These ideas should be familiar. However, we can now view these as ODEs. For example, if the velocity is 3 mph, then \(v(t) = 3\) can be written as the ODE \(p'(t) = 3\text{,}\) which is easily solved by integrating the right-hand side.
    🔗
    See if you can solve the following example.
    🔗

    Example 1.2.2. Finding Position from Acceleration.

    Find the position function \(p(t)\) of a moving object given:
    \begin{equation*} a(t) = 2t + 1, \quad v(0) = -7, \quad p(0) = 5. \end{equation*}
    🔗
    Solution.
    Since \(a = v'\text{,}\) we integrate \(a(t)\) to find \(v(t)\text{.}\)
    🔗
    First integration: Starting from \(v(t) = \int (2t+1)\, dt\text{:}\)
    🔗
    Solution 1. Reveal
    \begin{equation*} v(t) = t^2 + t + C. \end{equation*}
    Applying \(v(0) = -7\text{:}\)
    \begin{equation*} -7 = 0 + 0 + C \implies C = -7, \end{equation*}
    so \(v(t) = t^2 + t - 7\text{.}\)
    🔗
    🔗
    Since \(v = p'\text{,}\) we integrate \(v(t)\) to find \(p(t)\text{.}\)
    🔗
    Second integration: Starting from \(p(t) = \int (t^2 + t - 7)\, dt\text{:}\)
    🔗
    Solution 2. Reveal
    \begin{equation*} p(t) = \frac{1}{3}t^3 + \frac{1}{2}t^2 - 7t + C. \end{equation*}
    Applying \(p(0) = 5\text{:}\)
    \begin{equation*} 5 = C \implies C = 5, \end{equation*}
    so the specific solution is:
    \begin{equation*} p(t) = \frac{1}{3}t^3 + \frac{1}{2}t^2 - 7t + 5. \end{equation*}
    🔗
    🔗
    🔗
    🔗

    Example 1.2.3. Braking Distance.

    The brakes of a car travelling at \(70\) mph give a constant deceleration of \(10\) mi/h\(^2\text{.}\) How long, in hours, until the car comes to a full stop?
    🔗
    Hint.
    If there is a constant deceleration of \(10\text{,}\) then \(a(t)=-10\text{.}\) Solve for the velocity to answer the question.
    🔗
    🔗
    Solution.
    We are given \(a(t) = -10\) and \(v(0) = 70\text{.}\) Integrating:
    \begin{equation*} v(t) = \int -10\, dt = -10t + C. \end{equation*}
    Applying \(v(0) = 70\text{:}\)
    \begin{equation*} C = 70 \implies v(t) = -10t + 70. \end{equation*}
    The car stops when \(v(t) = 0\text{:}\)
    \begin{equation*} 0 = -10t + 70 \implies t = 7 \text{ hours.} \end{equation*}
    🔗
    Note: be careful about units! The deceleration is given in mi/h\(^2\text{,}\) so the answer is in hours.
