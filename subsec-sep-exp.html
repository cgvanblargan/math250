    Section 1.4.3 Separation of Variables: Important Example with Exponentials

    Recall the following population modelling ODE: \(\frac{dP}{dt} = 2P\text{,}\) which says the population grows at a rate twice the current population. We were previously able to guess and check some solutions to this. With separation of variables we will know, without a doubt, what all the solutions are.
    🔗
    First, why is this ODE separable? Recall that a separable ODE is one that can be written in the form:
    \begin{equation*} \frac{dy}{dx} = f(y)g(x), \end{equation*}
    or with our current variables:
    \begin{equation*} \frac{dP}{dt} = f(P)g(t). \end{equation*}
    But looking at \(\frac{dP}{dt} = 2P\text{,}\) there doesn’t seem to be any \(g(t)\) function! Think about what you could set \(f(P)\) and \(g(t)\) to be.
    🔗
    Solution. Why is this separable?
    Functions can be constant! So \(f\) and \(g\) may end up being constant functions, where there is no explicit mention of the independent variable. In this case there are two natural choices:
    1. \(f(P) = 2P\) and \(g(t) = 1\)
      🔗
    2. \(f(P) = P\) and \(g(t) = 2\)
      🔗
    Both choices work and will ultimately give the same general solution. In the example below, we will use choice (1): \(f(P) = 2P\) and \(g(t) = 1\text{.}\)
    🔗
    🔗
    Now that we have established it is separable, let’s use separation of variables to solve it! Try as much as you can on your own, and check with the solution as you go. In this problem, there are some subtle things we take note of in step 4 when solving for \(y\text{.}\)
    🔗

    Example 1.4.1. An important observation with exponentials.

    Find the general solution to \(\frac{dy}{dx} = 2y\text{.}\)
    🔗
    Hint.
    Follow the same five steps as in the previous example:
    1. Find equilibrium solutions.
      🔗
    2. Separate the variables.
      🔗
    3. Integrate both sides.
      🔗
    4. Solve for \(y\text{.}\)
      🔗
    5. Write the general solution.
      🔗
    🔗
    🔗
    Solution.
    Step 1: Find equilibrium solutions.
    🔗
    The only equilibrium solution is \(y = 0\text{.}\)
    🔗
    Solution 1. Work to solve for the equilibrium solution
    In separable ODEs, we can always find the equilibrium solution by solving \(f(y) = 0\text{.}\) Here \(f(y) = 2y\text{,}\) so
    \begin{equation*} 2y = 0 \implies y = 0. \end{equation*}
    So \(y = 0\) is the only equilibrium solution.
    🔗
    🔗
    Steps 2 and 3: Separate the variables and integrate.
    🔗
    You should arrive at \(\ln|y| = 2x + C\text{.}\) Nothing is too interesting in these two steps, but you can see the work below if you are unsure.
    🔗
    Solution 2. Work for steps 2 and 3
    Assuming \(y \neq 0\text{,}\) divide both sides by \(y\) and multiply by \(dx\text{:}\)
    \begin{equation*} \frac{1}{y}\, dy = 2\, dx. \end{equation*}
    🔗
    Then we integrate:
    \begin{equation*} \int \frac{1}{y}\, dy = \int 2\, dx \implies \ln|y| = 2x + C. \end{equation*}
    🔗

    Note 1.4.2.

    Don’t forget the absolute value in \(\ln|y|\text{!}\)
    🔗
    🔗
    🔗
    Step 4: Solve for \(y\text{.}\)
    🔗
    Starting from \(\ln|y| = 2x + C\text{,}\) we find:
    \begin{equation*} |y| = e^C e^{2x}. \end{equation*}
    This step just involves standard exponential rules, so make sure you are comfortable with it! You can read the solution below if you are stuck.
    🔗
    Solution 3. Step 4a: Algebra to get \(|y| = e^C e^{2x}\text{.}\)
    Exponentiating both sides
    \begin{equation*} e^{\ln|y|} = e^{2x+C} \end{equation*}
    \begin{equation*} |y| = e^{2x+C} \end{equation*}
    and applying \(e^{a+b} = e^a e^b\text{:}\)
    \begin{equation*} |y| = e^{2x+C} = e^C e^{2x}. \end{equation*}
    🔗
    🔗
    Finally, we get to the interesting part of this question! Solving for \(y\) here in a nice form takes some work. But the result of this will be a big takeaway for solving similar questions with exponentials. Do your best to solve for \(y\) here, and then definitely read solution below!!
    🔗
    Solution 4. Step 4b: Removing the absolute value and simplifying the constant
    The steps involved here involve careful reasoning about the constant and exponential function. Do not fret, we wil summarize this all at the end of the page and make sure you are confident with similar questions going forward!
    🔗
    We are starting from: \(|y| = e^C e^{2x}\text{.}\) Removing the absolute value introduces a \(\pm\text{:}\)
    \begin{equation*} y = \pm e^C e^{2x}. \end{equation*}
    As \(C\) varies over all real numbers, \(e^C\) ranges over all positive numbers. That is, every positive number can be written as \(e^C\) for some \(C\text{.}\) Therefore \(-e^C\) attains all negative numbers. Together, \(\pm e^C\) ranges over all nonzero real numbers. So we replace \(\pm e^C\) with a new constant \(C\) varying over all nonzero reals:
    \begin{equation*} y = Ce^{2x}, \quad C \neq 0. \end{equation*}
    🔗
    🔗
    Step 5: Write the general solution.
    🔗
    From Steps 1 and 4, we have found:
    \begin{equation*} y = 0 \quad \text{or} \quad y = Ce^{2x}, \quad C \neq 0. \end{equation*}
    Notice that setting \(C = 0\) in \(y = Ce^{2x}\) gives \(y = 0\text{,}\) recovering the equilibrium solution. So the general solution can be written compactly as:
    \begin{equation*} y = Ce^{2x}, \quad C \text{ any real number.} \end{equation*}
    🔗
    🔗
    🔗
    Step 4 involved some subtle reasoning, but this pattern comes up often in problems involving exponentials. So going forward we can skip many of the details, as discussed in the remark below.
    🔗

    Remark 1.4.4. A Useful Shortcut for Exponential Solutions.

    In particular, whenever you arrive at an expression of the form
    \begin{equation*} \ln|y| = h(x) + C, \quad C \text{ any real number,} \end{equation*}
    it always results in
    \begin{equation*} y = Ce^{h(x)}, \quad C \text{ any nonzero real number.} \end{equation*}
    🔗
    Even more, this typically arises after applying separation of variables to an ODE of the form \(\frac{dy}{dx} = yg(x)\text{,}\) where the only equilibrium solution is \(y = 0\text{.}\) So the general solution — combining the equilibrium solution with the explicit solution — can always be written compactly as:
    \begin{equation*} y = Ce^{h(x)}, \quad C \text{ any real number.} \end{equation*}
    🔗
    🔗
    The reasoning in Step 4 will come up again and again whenever you solve an ODE with an exponential solution. Getting comfortable with it now will save you a lot of trouble going forward! Let’s quickly drive this point home with the following question.
    🔗

    Checkpoint 1.4.5.

    Your friend is solving the ODE \(\frac{dy}{dt} = yx\) and has correctly found:
    1. \(y = 0\) is the only equilibrium solution.
      🔗
    2. Separation of variables yields \(|y| = e^C e^{x^2/2}\text{.}\)
      🔗
    They need your help finishing! Which of the following is the correct general solution?
    🔗
    • \(y = Ce^{x^2/2}\text{,}\) where \(C\) is any real number.
    • Exactly! \(\pm e^C\) ranges over all nonzero reals, and combining with the equilibrium solution \(y = 0\) means \(C\) can be any real number.
    • \(y = Ce^{x^2/2}\text{,}\) where \(C\) is any nonzero real number.
    • Close — but don’t forget the equilibrium solution \(y = 0\text{!}\) Setting \(C = 0\) recovers it, so \(C\) can be any real number.
    • \(y = e^C e^{x^2/2}\text{.}\)
    • Not quite — \(e^C\) is always positive, so this misses all the negative solutions. Don’t forget the \(\pm\) when removing the absolute value!
    • \(y = e^C e^{x^2/2}\) or \(y = 0\text{.}\)
    • You remembered the equilibrium solution, but \(e^C\) is always positive — this still misses all the negative solutions. The \(\pm\) is needed when removing the absolute value!
    🔗
