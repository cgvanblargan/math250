    Section 2.2.5 Applying Phase Diagrams to Population Models

    We now apply the phase diagram method to answer real questions about population models. The key question we focus on is: for which initial values \(P_0 = P(0)\) does a population grow forever?
    🔗

    Activity 2.2.1. Example 1.

    Suppose a population is modelled by
    \begin{equation*} \frac{dP}{dt} = (P-2)(P-4), \quad P(0) = P_0. \end{equation*}
    Find all values of \(P_0\) for which the population grows forever.
    🔗

    (a) Step 1: Find the equilibrium solutions.

    Find all equilibrium solutions to \(\dfrac{dP}{dt} = (P-2)(P-4)\text{.}\)
    🔗
    Solution. Solution
    Set \((P-2)(P-4) = 0\text{,}\) giving \(P = 2\) and \(P = 4\text{.}\)
    🔗
    🔗
    🔗

    (b) Step 2: Draw the phase diagram.

    Determine the sign of \(\dfrac{dP}{dt}\) in each region and draw the phase diagram.
    🔗
    Solution. Solution
    Test a value in each region:
    • \(P = 0\text{:}\) \((0-2)(0-4) = (-2)(-4) = 8 \gt 0\text{,}\) so arrow points right.
      🔗
      🔗
    • \(P = 3\text{:}\) \((3-2)(3-4) = (1)(-1) = -1 \lt 0\text{,}\) so arrow points left.
      🔗
      🔗
    • \(P = 5\text{:}\) \((5-2)(5-4) = (3)(1) = 3 \gt 0\text{,}\) so arrow points right.
      🔗
      🔗
    🔗
    🔗
    🔗

    (c) Step 3: Determine which \(P_0\) give growth forever.

    Using the phase diagram, for which values of \(P_0\) does \(P(t)\) grow forever?
    🔗
    • \(P_0 \in (-\infty, 2) \cup (4, \infty)\)
    • Correct! Both regions have rightward arrows, so solutions starting there are increasing and grow forever.
    • \(P_0 \in (2, 4)\)
    • Not quite. In this region the arrow points left, so solutions are decreasing, not growing.
    • \(P_0 \in (0, 2) \cup (4, \infty)\)
    • Close, but the left region should include all \(P \lt 2\text{,}\) not just \(P \in (0,2)\text{.}\) Solutions starting at negative values of \(P\) also have rightward arrows.
    • \(P_0 \gt 4\) only
    • Not quite. Check the arrow direction for \(P \lt 2\) as well — it also points right, so solutions there are also growing.
    🔗
    🔗

    Activity 2.2.2. Example 2.

    Suppose a population is modelled by
    \begin{equation*} \frac{dP}{dt} = 10P - P^2 = P(10-P), \quad P(0) = P_0. \end{equation*}
    Find all values of \(P_0\) for which the population grows forever.
    🔗

    (a) Step 1: Find the equilibrium solutions.

    Find all equilibrium solutions to \(\dfrac{dP}{dt} = P(10-P)\text{.}\)
    🔗
    Solution. Solution
    Set \(P(10-P) = 0\text{,}\) giving \(P = 0\) and \(P = 10\text{.}\)
    🔗
    🔗
    🔗

    (b) Step 2: Draw the phase diagram.

    Determine the sign of \(\dfrac{dP}{dt}\) in each region and draw the phase diagram.
    🔗
    Solution. Solution
    Test a value in each region:
    • \(P = -1\text{:}\) \((-1)(11) = -11 \lt 0\text{,}\) so arrow points left.
      🔗
      🔗
    • \(P = 5\text{:}\) \((5)(5) = 25 \gt 0\text{,}\) so arrow points right.
      🔗
      🔗
    • \(P = 15\text{:}\) \((15)(-5) = -75 \lt 0\text{,}\) so arrow points left.
      🔗
      🔗
    🔗
    🔗
    🔗

    (c) Step 3: Determine which \(P_0\) give growth forever.

    Using the phase diagram, for which values of \(P_0\) does \(P(t)\) grow forever?
    🔗
    • \(P_0 \in (0, 10)\)
    • Correct! This is the only region with a rightward arrow, so solutions starting there are increasing and grow forever.
      By the way, \(P = 10\) is a stable equilibrium above, so the solution is trapped below it and converges to \(10\text{,}\) but is increasing for all of time.
    • \(P_0 \in (-\infty, 0) \cup (10, \infty)\)
    • Not quite. In both of these regions the arrow points left, so solutions are decreasing, not growing.
    • \(P_0 \gt 10\) only
    • Not quite. Check the arrow direction for \(P \gt 10\) — it points left, so solutions there are actually decreasing.
    • \(P_0 \in (0, \infty)\)
    • Close, but check the arrow direction for \(P \gt 10\) — it points left, so solutions starting above \(10\) are decreasing, not growing.
    🔗
