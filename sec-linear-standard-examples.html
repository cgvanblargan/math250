    Section 1.5.3 First-Order Linear ODEs: Standard Examples

    Okay so we have figured out that we have some sort of magical integrating factor to help us solve first-order linear ODEs. It’s time to make sure we understand how to solve problems with it! In this section we walk through two examples that highlight techniques that come up again and again when solving first-order linear ODEs. For the first example, follow along and work out each step as we go. The second example will get you trying much of it on your own!
    🔗

    Example 1.5.1. Solve \(y' + 2xy + x = 0\).

    Solve the first-order linear ODE \(y' + 2xy + x = 0\text{.}\)
    🔗
    Hint.
    • Rewrite in standard form \(y' + P(x)y = Q(x)\) by moving the \(x\) term to the right-hand side.
      🔗
      🔗
    • Compute the integrating factor \(\rho(x) = e^{\int P(x)\,dx}\text{.}\)
      🔗
      🔗
    • Multiply both sides by \(\rho(x)\text{,}\) recognize the left-hand side as a single derivative, then integrate both sides.
      🔗
      🔗
    🔗
    Solution.
    Step 1: Standard form. Move \(x\) to the right-hand side:
    \begin{equation*} y' + 2xy = -x. \end{equation*}
    So \(P(x) = 2x\) and \(Q(x) = -x\text{.}\)
    🔗
    Step 2: Integrating factor.
    \begin{equation*} \rho(x) =e^{\int P(x)\,dx}= e^{\int 2x\,dx} = e^{x^2}. \end{equation*}
    🔗
    Step 3: Multiply both sides by \(\rho(x)\text{.}\)
    \begin{equation*} e^{x^2} y' + 2xe^{x^2} y = -xe^{x^2}. \end{equation*}
    The left-hand side is \((e^{x^2} y)'\) by the product rule, so we can write:
    \begin{equation*} (e^{x^2} y)' = -xe^{x^2}. \end{equation*}
    🔗
    Step 4: Evaluate the integral
    \begin{equation*} y \cdot e^{x^2} = \int -x e^{x^2}\,dx = -\frac{1}{2}e^{x^2} + C. \end{equation*}
    🔗
    Solution. U-substitution details
    Let \(u = x^2\text{,}\) so \(du = 2x\,dx\text{,}\) which means \(x\,dx = \frac{1}{2}du\text{.}\) Then:
    \begin{align*} \int -x e^{x^2}\,dx \amp= \int -e^u \cdot \frac{1}{2}\,du\\ \amp= -\frac{1}{2}\int e^u\,du\\ \amp= -\frac{1}{2}e^u + C\\ \amp= -\frac{1}{2}e^{x^2} + C. \end{align*}
    🔗
    🔗
    Step 5: Solve for \(y\text{.}\) Divide both sides by \(e^{x^2}\text{:}\)
    \begin{equation*} y = \frac{-\frac{1}{2}e^{x^2} + C}{e^{x^2}} = -\frac{1}{2} + Ce^{-x^2}. \end{equation*}
    🔗
    🔗
    🔗

    Observation 1.5.2. A Shortcut.

    Look at what just happened in the previous example. Once we had \(\rho(x) = e^{x^2}\text{,}\) we multiplied through, recognized the left-hand side as \((e^{x^2} y)'\) by the product rule, integrated both sides, and solved for \(y\text{.}\) Those intermediate steps will always play out the same way, as the integrating factor is defined precisely to make the left-hand side collapse like that every single time.
    🔗
    So going forward, once you have \(\rho(x)\text{,}\) you can skip straight from the original ODE to
    \begin{equation*} y \cdot \rho(x) = \int \rho(x)\,Q(x)\,dx \end{equation*}
    without writing out the multiplication, product rule collapse, and integration steps in between. You will see this in action in the activity below.
    🔗
    🔗

    Activity 1.5.1. Practice: Solve \(y' + \frac{2}{x}y = 5x^2\).

    Try each part on your own before checking the solution.
    🔗

    (a) Finding the Integrating Factor.

    For the ODE \(y' + \dfrac{2}{x}y = 5x^2\text{,}\) we have \(P(x) = \dfrac{2}{x}\text{.}\) Which of the following correctly gives the integrating factor \(\rho(x) = e^{\int P(x)\,dx}\) with the integral evaluated?
    🔗
    • \(\rho(x) = e^{2\ln|x|}\)
    • Correct! We compute \(\int \frac{2}{x}\,dx = 2\ln|x|\text{,}\) so \(\rho(x) = e^{2\ln|x|}\text{.}\) We will simplify this in the next part.
    • \(\rho(x) = e^{2/x}\)
    • Not quite -- this skips the integral. We need \(e^{\int (2/x)\,dx}\text{,}\) not \(e^{P(x)}\text{.}\) Make sure to integrate \(P(x)\) first.
    • \(\rho(x) = e^{2x}\)
    • This would be the integrating factor if \(P(x) = 2\text{,}\) but here \(P(x) = \frac{2}{x}\text{.}\) Remember: \(\int \frac{2}{x}\,dx \ne 2x\text{.}\)
    • \(\rho(x) = e^{\ln|x|}\)
    • Close -- \(\int \frac{1}{x}\,dx = \ln|x|\text{,}\) but there is a factor of \(2\) in front. So \(\int \frac{2}{x}\,dx = 2\ln|x|\text{.}\)
    🔗

    (b) Simplifying the Integrating Factor.

    From the previous part, \(\rho(x) = e^{2\ln|x|}\text{.}\) Simplify this completely. Which of the following is correct?
    🔗
    • \(\rho(x) = x^2\)
    • Correct! Using the log property \(e^{a\ln|x|} = |x|^a\text{,}\) we get \(e^{2\ln|x|} = |x|^2 = x^2\text{,}\) since \(x^2\) is always non-negative and the absolute value drops.
    • \(\rho(x) = 2x\)
    • Not quite. Recall that \(e^{2\ln|x|} = e^{\ln(x^2)} = x^2\text{,}\) not \(2x\text{.}\) The exponent rule here is \(e^{a\ln|x|} = |x|^a\text{.}\)
    • \(\rho(x) = |x|\)
    • This would follow from \(e^{\ln|x|}\text{,}\) but we have \(e^{2\ln|x|}\text{.}\) The factor of \(2\) becomes the exponent on \(|x|\text{,}\) giving \(|x|^2 = x^2\text{.}\)
    • \(\rho(x) = e^{x^2}\)
    • This is the integrating factor from Example 1, not this problem. Here the exponent is \(2\ln|x|\text{,}\) and \(e^{2\ln|x|} = x^2\text{.}\)
    🔗

    (c) Finishing the Solution.

    From the previous parts: the ODE is \(y' + \dfrac{2}{x}y = 5x^2\) and \(\rho(x) = x^2\text{.}\) Using the shortcut
    \begin{equation*} y \cdot \rho(x) = \int \rho(x)\,Q(x)\,dx, \end{equation*}
    finish solving for \(y\text{.}\)
    🔗
    Solution.
    Substituting \(\rho(x) = x^2\) and \(Q(x) = 5x^2\text{:}\)
    \begin{align*} y \cdot \rho(x) \amp= \int \rho(x)\,Q(x)\,dx,\\ y \cdot x^2 \amp= \int 5x^4\,dx\\ y \cdot x^2 \amp= x^5 + C\\ y \amp= x^3 + Cx^{-2}. \end{align*}
    🔗
    🔗
    🔗
    The general solution is \(y = x^3 + Cx^{-2}\text{.}\) The key steps were identifying \(P(x) = \frac{2}{x}\text{,}\) computing and simplifying the integrating factor to \(\rho(x) = x^2\text{,}\) and then applying the shortcut to jump directly to integrating the right-hand side.
    🔗
    🔗

    Remark 1.5.3. Where Does \(C\) Go?

    In the last step above, we had \(x^2 y = x^5 + C\) and divided both sides by \(x^2\text{.}\) This gives
    \begin{equation*} y = x^3 + Cx^{-2}, \end{equation*}
    not \(y = x^3 + C\text{.}\) The constant \(C\) appears when you integrate the right-hand side, and it must be divided by \(\rho(x) = x^2\) along with everything else in that last step. Forgetting to carry the \(x^{-2}\) through to the \(C\) term is one of the most common mistakes in this method, so be careful!
