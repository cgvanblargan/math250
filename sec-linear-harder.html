    Section 1.5.4 First-Order Linear ODEs: A Harder Example

    Now for some practice solving a first-order linear ODE. This will help reinforce all the ideas we just went through, plus we will be getting our hands a little messier!
    🔗

    Activity 1.5.1. Practice: Solve \((x^4+5)y' + x^3 y = x^3\).

    In this activity we work through solving a first-order line ODE that is slightly more involved than the previous examples The first few parts will reinforce steps we already discussed, so this will be a great way to practice these steps. The only big difference is that in the last step, where the final integral is a little messier.
    🔗

    (a) Identifying \(P(x)\) and \(Q(x)\).

    Consider the ODE
    \begin{equation*} (x^4+5)y' + x^3 y = x^3. \end{equation*}
    Which of the following correctly identifies \(P(x)\) and \(Q(x)\) when this equation is written in standard form?
    🔗
    • \(P(x) = \dfrac{x^3}{x^4+5}\) and \(Q(x) = \dfrac{x^3}{x^4+5}\)
    • Correct! In standard form \(y' + P(x)y = Q(x)\text{:}\) \(y' + \dfrac{x^3}{x^4+5}y = \dfrac{x^3}{x^4+5}\text{.}\) Dividing both sides by \((x^4+5)\) gives \(P(x) = \frac{x^3}{x^4+5}\) and \(Q(x) = \frac{x^3}{x^4+5}\text{.}\)
    • \(P(x) = x^3\) and \(Q(x) = x^3\)
    • Not quite. In standard form \(y' + P(x)y = Q(x)\text{:}\) \(y' + \dfrac{x^3}{x^4+5}y = \dfrac{x^3}{x^4+5}\text{.}\) The coefficient on \(y'\) must be 1, so you need to divide every term by \((x^4+5)\) first.
    • \(P(x) = \dfrac{x^3}{x^4+5}\) and \(Q(x) = x^3\)
    • Close, but not quite. In standard form \(y' + P(x)y = Q(x)\text{:}\) \(y' + \dfrac{x^3}{x^4+5}y = \dfrac{x^3}{x^4+5}\text{.}\) \(P(x)\) is correct, but \(Q(x)\) needs to be divided by \((x^4+5)\) too -- every term gets divided.
    • \(P(x) = x^3\) and \(Q(x) = \dfrac{x^3}{x^4+5}\)
    • Close, but not quite. In standard form \(y' + P(x)y = Q(x)\text{:}\) \(y' + \dfrac{x^3}{x^4+5}y = \dfrac{x^3}{x^4+5}\text{.}\) \(Q(x)\) is correct, but \(P(x)\) also needs to be divided by \((x^4+5)\text{.}\)
    🔗

    (b) Finding the Integrating Factor.

    Now that we have \(P(x) = \dfrac{x^3}{x^4+5}\text{,}\) find the integrating factor \(\rho(x).\) Stop after integrating (the next part will simplify \(\rho\) further).
    🔗
    Hint.
    • Recall that \(\rho(x) = e^{\int P(x)\,dx}\text{.}\)
      🔗
      🔗
    • To evaluate \(\int \frac{x^3}{x^4+5}\,dx\) use a u-sub (what should u be?)
      🔗
      🔗
    • Don’t simplify the resulting \(\rho(x)\) yet, that’s the next part.
      🔗
      🔗
    🔗
    Solution.
    We compute \(\int P(x)\,dx = \int \frac{x^3}{x^4+5}\,dx\) using a u-substitution with \(u = x^4 + 5\text{.}\)
    🔗
    Solution. U-substitution details (read if stuck)
    Let \(u = x^4 + 5\text{,}\) so \(du = 4x^3\,dx\text{,}\) which means \(x^3\,dx = \frac{1}{4}du\text{.}\) Then:
    \begin{align*} \int \frac{x^3}{x^4+5}\,dx \amp= \int \frac{1}{u} \cdot \frac{1}{4}\,du\\ \amp= \frac{1}{4}\int \frac{1}{u}\,du\\ \amp= \frac{1}{4}\ln|u| + C\\ \amp= \frac{1}{4}\ln(x^4+5). \end{align*}
    (We drop the absolute value since \(x^4 + 5 \gt 0\) always, and drop \(C\) since we only need one antiderivative.)
    🔗
    🔗
    So,\(\int P(x)\,dx= \frac{1}{4}\ln(x^4+5)\text{,}\) meaning the integrating factor is
    \begin{equation*} \rho(x) = e^{\frac{1}{4}\ln(x^4+5)}. \end{equation*}
    🔗
    🔗
    🔗

    (c) Simplifying the Integrating Factor.

    From the previous part, \(\rho(x) = e^{\frac{1}{4}\ln(x^4+5)}\text{.}\) Which of the following correctly simplifies this?
    🔗
    • \(\rho(x) = (x^4+5)^{1/4}\)
    • Correct! Using the log property \(e^{a\ln(u)} = u^a\text{,}\) we get \(e^{\frac{1}{4}\ln(x^4+5)} = (x^4+5)^{1/4}\text{.}\)
    • \(\rho(x) = \frac{1}{4}(x^4+5)\)
    • Not quite. The \(\frac{1}{4}\) is in the exponent of \(e\text{,}\) not a coefficient out front. Use the property \(e^{a\ln(u)} = u^a\) with \(a = \frac{1}{4}\text{.}\)
    • \(\rho(x) = (x^4+5)^4\)
    • Close in structure, but the exponent should be \(\frac{1}{4}\text{,}\) not \(4\text{.}\) We have \(e^{\frac{1}{4}\ln(x^4+5)} = (x^4+5)^{1/4}\text{.}\)
    • \(\rho(x) = e^{x^4+5}\)
    • Remember that \(e^{\frac{1}{4}\ln(x^4+5)} \ne e^{x^4+5}\text{.}\) Use the log property \(e^{a\ln(u)} = u^a\) to simplify.
    🔗

    (d) Finishing the Solution.

    The next steps -- multiplying by \(\rho(x)\text{,}\) recognizing the product rule collapse, and integrating -- always work out the same way. So we can always jump directly to:
    \begin{equation*} \rho(x) \cdot y = \int \rho(x)\, Q(x)\,dx. \end{equation*}
    Fill in \(\rho(x)\) and \(Q(x)\) and finish solving for \(y\text{.}\)
    🔗
    Hint.
    • Substitute \(\rho(x) = (x^4+5)^{1/4}\) and \(Q(x) = \dfrac{x^3}{x^4+5}\) into \(\rho(x) \cdot y = \int \rho(x)\,Q(x)\,dx\) to get your first line.
      🔗
      🔗
    • Simplify the integrand -- notice that \((x^4+5)^{1/4} \cdot \dfrac{x^3}{x^4+5} = x^3(x^4+5)^{-3/4}\text{.}\) Then use the substitution \(u = x^4+5\text{.}\)
      🔗
      🔗
    🔗
    Solution.
    Substituting in \(\rho(x)\) and \(Q(x)\text{:}\)
    \begin{align*} \rho(x) \cdot y \amp= \int \rho(x)\, Q(x)\,dx\\ (x^4+5)^{1/4}\, y \amp= \int (x^4+5)^{1/4} \cdot \frac{x^3}{x^4+5}\,dx\\ (x^4+5)^{1/4}\, y \amp= \int x^3(x^4+5)^{-3/4}\,dx. \end{align*}
    🔗
    Solution. U-substitution details (read if stuck)
    Let \(u = x^4+5\text{,}\) so \(du = 4x^3\,dx\text{,}\) meaning \(x^3\,dx = \frac{1}{4}du\text{.}\) Then:
    \begin{align*} \int x^3(x^4+5)^{-3/4}\,dx \amp= \frac{1}{4}\int u^{-3/4}\,du\\ \amp= \frac{1}{4} \cdot \frac{u^{1/4}}{1/4} + C\\ \amp= u^{1/4} + C\\ \amp= (x^4+5)^{1/4} + C. \end{align*}
    🔗
    🔗
    So we have:
    \begin{align*} (x^4+5)^{1/4}\, y \amp= (x^4+5)^{1/4} + C\\ y \amp= 1 + C(x^4+5)^{-1/4}. \end{align*}
    🔗
    🔗
    🔗
    🔗
    Let’s recap the steps for this problem:
    🔗
    • Step 1. Divide through by \((x^4+5)\) to get standard form, giving \(P(x) = \dfrac{x^3}{x^4+5}\) and \(Q(x) = \dfrac{x^3}{x^4+5}\text{.}\)
      🔗
      🔗
    • Step 2. Find \(\rho(x) = e^{\int P(x)\,dx}\text{.}\) The integral required a u-substitution, giving \(e^{\frac{1}{4}\ln(x^4+5)}\text{,}\) which simplifies via log rules to \((x^4+5)^{1/4}\text{.}\)
      🔗
      🔗
    • Step 3. Jump directly to \(\rho(x) \cdot y = \int \rho(x)\,Q(x)\,dx\) and evaluate the right-hand side, which required another u-substitution.
      🔗
      🔗
    • Step 4. Divide both sides by \(\rho(x)\) to solve for \(y\text{.}\) Remember that \(C\) gets divided by \(\rho(x)\text{,}\) too! The final answer is \(y=1+C(x^4+5)^{-1/4}\text{,}\) not \(y=1+C\text{.}\)
      🔗
      🔗
    🔗
