    Section 1.5.1 Definition of 1st Order Linear ODEs

    Not all first-order ODEs are separable! Another important class is first-order linear equations. Before we can do anything interesting with them, we first need to recognize what they look like.
    🔗

    Definition 1.5.1. First-Order Linear Differential Equation.

    A first-order linear differential equation is one that can be written in the form
    \begin{equation*} \frac{dy}{dx} + P(x)y = Q(x), \end{equation*}
    where \(P(x)\) and \(Q(x)\) are functions of \(x\) only.
    🔗
    🔗
    As quick example, the following ODE is first-order linear:
    🔗
    \begin{equation*} \frac{dy}{dx} + \underbrace{\sin(x)}_{P(x)}\, y = \underbrace{\cos(x)}_{Q(x)} \end{equation*}
    The key idea here is that the equation is linear in \(y\) and \(y'\text{:}\) both \(y\) and \(y'\) appear only to the first power, and never multiplied together. In particular, notice:
    🔗
    • No products or powers of \(y'\) and \(y\) -- terms like \(y' \cdot y\) or \(y^2\) are not allowed.
      🔗
      🔗
    • No compositions involving \(y\) -- terms like \(\cos(y)\) or \(e^y\) are not allowed.
      🔗
      🔗
    • The functions \(P(x)\) and \(Q(x)\) can be anything -- \(\sin(x)\text{,}\) \(x^2\text{,}\) constants, and so on -- as long as they depend only on \(x\text{,}\) not on \(y\text{.}\)
      🔗
      🔗
    Let’s explore these ideas in the following non-examples.
    🔗
    • \(\dfrac{dy}{dx} + x\sin(y) = \cos(x)\text{.}\) This is not linear: the term \(\sin(y)\) is a composition involving \(y\text{,}\) so this cannot be written in the form \(\frac{dy}{dx} + P(x)y = Q(x)\text{.}\)
      🔗
      🔗
    • \(y'y = x\text{.}\) This is not linear: the left side is a product of \(y'\) and \(y\text{,}\) which is not allowed.
      🔗
      🔗
    Just as with separable ODEs, it is not always immediately obvious whether an ODE is linear. We will see this in the following example.
    🔗

    Example 1.5.2. Rewriting in Standard Form.

    Consider the equation \((x^2 + 1)y' + y + x = 0\text{.}\) Is this linear? If so, identify \(P(x)\) and \(Q(x)\text{.}\)
    🔗
    Hint.
    Manipulate the ODE to get it in the form \(\dfrac{dy}{dx} + P(x)y = Q(x)\text{.}\)
    🔗
    🔗
    Solution.
    First move the \(x\) term to the right:
    \begin{equation*} (x^2 + 1)y' + y = -x. \end{equation*}
    Now divide through by \((x^2 + 1)\text{:}\)
    \begin{equation*} y' + \frac{1}{x^2 + 1}y = \frac{-x}{x^2 + 1}. \end{equation*}
    This is in standard form with
    \begin{equation*} P(x) = \frac{1}{x^2 + 1}, \qquad Q(x) = \frac{-x}{x^2 + 1}. \end{equation*}
    So yes, this equation is linear!
    🔗
    🔗
    🔗
    Now just a quick check before we get to the fun part!!
    🔗

    Checkpoint 1.5.3. Checkpoint: Identifying Linear Equations.

    Which of the following are first-order linear differential equations? Select all that apply.
    🔗
    • \(\dfrac{dy}{dx} + x^2 y = e^x\)
    • Linear. This is already in standard form with \(P(x) = x^2\) and \(Q(x) = e^x\text{,}\) both functions of \(x\) only.
    • \(xy' + 2y = \cos(x)\)
    • Linear. Dividing through by \(x\) gives \(y' + \frac{2}{x}y = \frac{\cos(x)}{x}\text{,}\) which is in standard form with \(P(x) = \frac{2}{x}\) and \(Q(x) = \frac{\cos(x)}{x}\text{.}\)
    • \(\dfrac{dy}{dx} + y^2 = x\)
    • Not linear. The term \(y^2\) is a power of \(y\) greater than one, which is not allowed in a linear equation.
    • \(y' = \sin(y) + x\)
    • Not linear. The term \(\sin(y)\) is a composition involving \(y\text{,}\) so this cannot be written in the form \(\frac{dy}{dx} + P(x)y = Q(x)\text{.}\)
    • \((x+1)y' - 3y = x^2\)
    • Linear. Dividing through by \((x+1)\) gives \(y' - \frac{3}{x+1}y = \frac{x^2}{x+1}\text{,}\) which is in standard form with \(P(x) = -\frac{3}{x+1}\) and \(Q(x) = \frac{x^2}{x+1}\text{.}\)
    🔗
