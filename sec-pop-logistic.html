    Section 2.1.2 Solving the Logistic Equation

    In the previous page we set up the logistic equation \(\dfrac{dP}{dt} = kP(M - P)\) as a model for population growth that accounts for crowding and resource limits. Now let’s actually solve it. The algebra is a workout, so we have organized things so you can read the big picture first and dig into the details later when you need them for homework.
    🔗

    Example 2.1.1. Solving a Logistic Equation.

    The birth rate of a population \(P\) is the constant 20 and the death rate is proportional to \(P\) with proportionality constant 4. Set up and solve the resulting differential equation for \(P(t)\text{.}\)
    🔗
    Using \(\dfrac{dP}{dt} = \beta P - \alpha P\) with \(\beta = 20\) and \(\alpha = 4P\text{:}\)
    \begin{align*} \frac{dP}{dt} \amp= 20P - (4P)P = 4P(5 - P). \end{align*}
    This is the logistic equation with \(k = 4\) and \(M = 5\text{.}\)
    🔗
    Step 1: Equilibrium solutions. Setting \(4P(5 - P) = 0\) gives
    \begin{equation*} P(t) = 0 \qquad \text{and} \qquad P(t) = 5. \end{equation*}
    🔗
    Solution 1. Work for equilibrium solutions (if stuck)
    Constant solutions satisfy \(\dfrac{dP}{dt} = 0\text{.}\) So we solve
    \begin{equation*} 4P(5 - P) = 0, \end{equation*}
    which gives \(P = 0\) or \(5 - P = 0\text{,}\) that is \(P = 5\text{.}\)
    🔗
    🔗
    Step 2: Separate variables. For non-equilibrium solutions:
    \begin{equation*} \frac{1}{P(5 - P)}\, dP = 4\, dt. \end{equation*}
    🔗
    Step 3: Integrate both sides.
    \begin{equation*} \int \frac{1}{P(5-P)}\,dP = \int 4\,dt \end{equation*}
    gives
    \begin{equation*} \ln\left|\frac{P}{5-P}\right| = 20t + C, \qquad \text{so} \qquad \frac{P}{5-P} = Ce^{20t}. \end{equation*}
    🔗
    Solution 2. Full LHS integration with partial fractions (can skim for now, useful for homework)
    We use partial fractions. Write
    \begin{equation*} \frac{1}{P(5-P)} = \frac{A}{P} + \frac{B}{5-P}. \end{equation*}
    Multiplying both sides by \(P(5-P)\) and matching coefficients:
    \begin{equation*} 1 = A(5-P) + BP = (-A+B)P + 5A. \end{equation*}
    So \(A = \tfrac{1}{5}\) and \(B = \tfrac{1}{5}\text{.}\) Therefore:
    \begin{align*} \int \frac{1}{P(5-P)}\,dP \amp= \frac{1}{5}\int \frac{dP}{P} + \frac{1}{5}\int \frac{dP}{5-P}\\ \amp= \frac{1}{5}\ln|P| - \frac{1}{5}\ln|5-P|. \end{align*}
    Multiplying through by 5 and using log rules:
    \begin{equation*} \ln\left|\frac{P}{5-P}\right| = 20t + C. \end{equation*}
    Exponentiating: \(\dfrac{P}{5-P} = Ce^{20t}\text{.}\)
    🔗
    🔗
    Step 4: Solve for \(P(t)\text{.}\) The general solution is
    \begin{equation*} P(t) = \frac{5}{Ce^{-20t} + 1} \quad \text{or} \quad P(t) = 0, \end{equation*}
    where \(C\) is any real number. Note that \(C = 0\) recovers the equilibrium solution \(P(t) = 5\text{.}\) The equilibrium solution \(P(t) = 0\) is not captured by this formula and must be noted separately.
    🔗
    Solution 3. Algebra: solving for \(P\) explicitly (can skim for now)
    Starting from \(\dfrac{P}{5-P} = Ce^{20t}\text{,}\) take reciprocals of both sides:
    \begin{equation*} \frac{5-P}{P} = \frac{1}{C}e^{-20t}. \end{equation*}
    Split the left side:
    \begin{equation*} \frac{5}{P} - 1 = \frac{1}{C}e^{-20t}. \end{equation*}
    Relabeling \(\tfrac{1}{C}\) as \(C\text{:}\)
    \begin{equation*} \frac{5}{P} = Ce^{-20t} + 1. \end{equation*}
    Taking reciprocals:
    \begin{equation*} P(t) = \frac{5}{Ce^{-20t} + 1}. \end{equation*}
    🔗
    🔗
    🔗
    Now that we have an explicit formula for \(P(t)\text{,}\) one of the most important questions in population modelling is: what happens in the long run? Does the population grow forever, crash to zero, or stabilize at some value?
    🔗

    Checkpoint 2.1.2.

    In the solution \(P(t) = \dfrac{5}{Ce^{-20t} + 1}\text{,}\) what is \(\displaystyle\lim_{t \to \infty} P(t)\text{?}\)
    🔗
    • \(0\)
    • Not quite. Think about what happens to \(e^{-20t}\) as \(t \to \infty\text{.}\) It decays to zero, not grows. So the denominator does not blow up.
    • \(5\)
    • Correct! As \(t \to \infty\text{,}\) \(e^{-20t} \to 0\text{,}\) so \(Ce^{-20t} + 1 \to 1\text{,}\) giving \(P(t) \to \dfrac{5}{1} = 5\text{.}\)
    • \(\infty\)
    • Not quite. Since \(e^{-20t} \to 0\) as \(t \to \infty\text{,}\) the denominator approaches 1, not 0. So \(P(t)\) does not blow up.
    • It depends on \(C\text{.}\)
    • Good instinct, but notice that \(Ce^{-20t} \to 0\) as \(t \to \infty\) regardless of what \(C\) is. The limit is the same for every solution.
    🔗
    That limit equals exactly \(M = 5\text{,}\) the carrying capacity. This is not a coincidence. For the logistic equation with \(k \gt 0\text{,}\) every solution with \(P(0) \gt 0\) satisfies
    \begin{equation*} \lim_{t \to \infty} P(t) = M. \end{equation*}
    No matter where the population starts, it always levels off at the carrying capacity. Also notice that \(P(0) = 0\) we recover one of the equilibrium solutions: if the population starts at zero, it stays there. A similar thing happens for \(P(0) = 5\text{!}\) This is exactly the realistic, well-behaved growth we were hoping to model.
    🔗
    The logistic equation with \(k \gt 0\) is a beautifully behaved model. Solutions are bounded, they stabilize, and the long-term behavior is completely predictable regardless of initial conditions. In the next section we look at a similar model with \(k \lt 0\text{,}\) and we find much more pathological behavior: populations that go extinct in finite time and populations that explode to infinity in finite time.
