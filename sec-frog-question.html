    Section 2.2.6 Applying Phase Diagrams: The Frog Question

    We will do a similar example now as the previous page, except now we explore how the behavior of a solution will differ given a change in the initial value. If you are stuck, you can reference the previous page for some guidance!
    🔗

    Activity 2.2.1. The Mosquito Population.

    Suppose the population of mosquitoes in a pond is given by \(P(t)\text{,}\) modelled by the differential equation
    \begin{equation*} \frac{dP}{dt} = 4P(P - 30). \end{equation*}
    There are currently 100 mosquitoes in the pond, so \(P(0) = 100\text{.}\) You would like the population to decrease to zero over time.
    🔗

    (a) P(0)=100.

    Determine whether \(P(t)\) will decrease to zero when \(P(0) = 100\text{.}\)
    🔗
    Hint.
    First find the equilibrium solutions by setting \(4P(P-30) = 0\text{,}\) then draw the phase diagram. Where does \(P(0)\) lie?
    🔗
    🔗
    Solution.
    The equilibrium solutions are \(P = 0\) and \(P = 30\text{.}\) The phase diagram is:
    🔗
    Since \(P(0) = 100 \gt 30\text{,}\) the initial value lies in the region where the arrow points right. By the consequences of existence and uniqueness, \(P(t)\) is increasing for all time, so the population will not decrease to zero.
    🔗
    Solution. Sign analysis (if stuck on the phase diagram)
    Test a value in each region:
    • \(P = -1\text{:}\) \(4(-1)(-31) = 124 \gt 0\text{,}\) arrow right.
      🔗
      🔗
    • \(P = 15\text{:}\) \(4(15)(-15) = -900 \lt 0\text{,}\) arrow left.
      🔗
      🔗
    • \(P = 40\text{:}\) \(4(40)(10) = 1600 \gt 0\text{,}\) arrow right.
      🔗
      🔗
    🔗
    🔗
    🔗

    (b) The magic frog.

    A man offers a magic frog that can eat 80 mosquitoes instantly, reducing the population to \(P(0) = 20\text{.}\) Will the population now decrease to zero?
    🔗
    • Yes the population will decrease to zero.
    • Correct! With \(P(0) = 20\text{,}\) the initial value falls in the region \(0 \lt P \lt 30\text{,}\) where the arrow points left. By the consequences of existence and uniqueness, \(P(t)\) is decreasing for all time and trapped above the stable equilibrium \(P = 0\text{,}\) so it must converge to \(0\) as \(t \to \infty\text{.}\) The frog works!
    • No, the population will still increase.
    • Not quite. Check the phase diagram from part 1 again — which direction does the arrow point in the region \(0 \lt P \lt 30\text{?}\)
    • No, \(P = 0\) is unstable, so solutions won’t converge to it.
    • Check the arrow directions on both sides of \(P = 0\) in the phase diagram. Do they point toward it or away from it?
    • It depends on the value of \(k\) in the model.
    • The constant \(k = 4\) affects the speed of change but not the direction — that is determined entirely by the sign of \(f(P)\text{,}\) which the phase diagram captures. The phase diagram is all we need.
    🔗

    Key Takeaway: Phase Diagrams as a First Step

    When analyzing the behavior of solutions to a first-order autonomous ODE, such as whether they are increasing or decreasing, and what they converge to, a good first step is always to find the equilibrium solutions and draw the phase diagram, even if you are not explicitly asked to. All the information you need is in the diagram.
    🔗
    In this example, the phase diagram told us everything:
    • Part 1: \(P(0) = 100\) falls in a zone where the arrow points right, so \(P(t)\) stays increasing, and so it will not decrease to zero.
      🔗
      🔗
    • Part 2: \(P(0) = 20\) falls in a zone where the arrow points left toward the stable equilibrium \(P = 0\text{,}\) so \(P(t)\) is decreasing and converges to \(0\text{.}\)
