    Section 1.4.2 Separation of Variables

    The method we started exploring on the previous page is called separation of variables. Let’s now make it precise.
    🔗

    Definition 1.4.1. Separable Differential Equation.

    A separable differential equation is one that can be written in the form
    \begin{equation*} \frac{dy}{dx} = f(y)\,g(x). \end{equation*}
    It can be solved using separation of variables:
    \begin{equation*} \int \frac{1}{f(y)}\, dy = \int g(x)\, dx. \end{equation*}
    🔗
    This method captures all solutions that are not singular. When \(f(y) = 0\text{,}\) we cannot divide by \(f(y)\text{,}\) so separation of variables may miss some solutions, but any solution missed this way is precisely an equilibrium (constant) solution.
    🔗
    🔗
    A helpful (if informal) way to remember this is the following "fake proof":
    🔗
    Starting from \(\frac{dy}{dx} = f(y)g(x)\text{,}\) treat \(\frac{dy}{dx}\) as a fraction and divide both sides by \(f(y)\text{:}\)
    \begin{equation*} \frac{1}{f(y)}\,\frac{dy}{dx} = g(x). \end{equation*}
    Then "multiply both sides by \(dx\)" and integrate:
    \begin{equation*} \int \frac{1}{f(y)}\, dy = \int g(x)\, dx. \end{equation*}
    While this is not a rigorous proof, it is a reliable way to remember how to apply the method.
    🔗

    Precise Justification (optional — for the curious!).

    We want to rigorously justify why \(\int \frac{1}{f(y)}\, dy = \int g(x)\, dx\) follows from \(\frac{dy}{dx} = f(y)g(x)\text{.}\)
    🔗
    Let \(H(y)\) be an antiderivative of \(\frac{1}{f(y)}\text{,}\) so that \(H'(y) = \frac{1}{f(y)}\text{.}\) Consider the function \(H(y(x))\text{,}\) viewed as a function of \(x\text{.}\) By the chain rule:
    \begin{equation*} \frac{d}{dx}H(y(x)) = H'(y(x))\cdot \frac{dy}{dx} = \frac{1}{f(y)}\cdot f(y)g(x) = g(x). \end{equation*}
    So \(H(y(x))\) is an antiderivative of \(g(x)\) with respect to \(x\text{.}\) That is:
    \begin{equation*} H(y(x)) = \int g(x)\, dx + C. \end{equation*}
    But \(H(y(x)) = \int \frac{1}{f(y)}\, dy\) by definition of \(H\text{.}\) So we arrive at:
    \begin{equation*} \int \frac{1}{f(y)}\, dy = \int g(x)\, dx + C, \end{equation*}
    which is exactly what the method produces. The "fake proof" manipulation, while informal, gives the correct result.
    🔗
    🔗
    Notice that separation of variables misses any solution where \(f(y) = 0\text{,}\) since we cannot divide by zero in that step. It turns out these are precisely the equilibrium solutions -- the ones you already know how to find from Section 1.1.5!
    🔗
    Solution. Why does \(f(y) = 0\) give equilibrium solutions?
    Recall that an equilibrium solution is a constant solution \(y = C\text{.}\) If \(y\) is constant, then \(y' = 0\text{,}\) so the ODE \(\frac{dy}{dx} = f(y)g(x)\) becomes
    \begin{equation*} 0 = f(C)\,g(x). \end{equation*}
    For this to hold for all \(x\text{,}\) we need \(f(C) = 0\text{.}\) So the equilibrium solutions are exactly the values of \(y\) where \(f(y) = 0\text{.}\)
    🔗
    🔗

    Observation 1.4.2. General Solution of a Separable ODE.

    The general solution to \(\frac{dy}{dx} = f(y)\,g(x)\) consists of:
    1. All solutions found by separation of variables.
      🔗
      🔗
    2. All equilibrium solutions, which correspond to values of \(y\) where \(f(y) = 0\text{.}\)
      🔗
      🔗
    🔗
    🔗
    In the following example, we go through each step in applying separation of variables. At this point, you are still getting used to the process, so feel free to follow along step by step!
    🔗

    Example 1.4.3. Solving a Separable ODE.

    Find the general explicit solution to \(\frac{dy}{dx} = 3y^2 x^3\text{.}\)
    🔗
    Solution.
    Step 1: Find equilibrium solutions.
    🔗
    From the observation above, a shortcut to finding equilibrium solutions in separable ODEs is to solve \(f(y) = 0\text{.}\) Here \(f(y) = y^2\text{,}\) so
    \begin{equation*} y^2 = 0 \implies y = 0. \end{equation*}
    So \(y = 0\) is the only equilibrium solution.
    🔗
    Step 2: Separate the variables.
    🔗
    Assuming \(y \neq 0\text{,}\) divide both sides by \(y^2\) and "multiply by \(dx\)":
    \begin{equation*} \frac{1}{y^2}\, dy = 3x^3\, dx. \end{equation*}
    🔗
    Solution 1. Explanation: separating variables
    We start from \(\frac{dy}{dx} = 3y^2 x^3\text{.}\) Here \(f(y) = y^2\) and \(g(x) = 3x^3\text{.}\) Dividing both sides by \(f(y) = y^2\) and treating \(dx\) as we described:
    \begin{equation*} \frac{1}{y^2}\,\frac{dy}{dx} = 3x^3 \implies \frac{1}{y^2}\,dy = 3x^3\,dx. \end{equation*}
    🔗
    🔗
    Step 3: Integrate both sides.
    🔗
    \begin{align*} \int \frac{1}{y^2}\, dy \amp= \int 3x^3\, dx\\ -y^{-1} \amp= \frac{3}{4}x^4 + C \end{align*}
    (We place the constant \(C\) only on the right-hand side, since combining two constants of integration still gives a single arbitrary constant.)
    🔗
    Step 4: Solve for \(y\text{.}\)
    🔗
    From \(-\frac{1}{y} = \frac{3}{4}x^4 + C\text{,}\) taking reciprocals:
    \begin{equation*} y = \frac{-1}{\frac{3}{4}x^4 + C}. \end{equation*}
    🔗
    Solution 2. Full working
    \begin{align*} -\frac{1}{y} \amp= \frac{3}{4}x^4 + C\\ \frac{1}{y} \amp= -\frac{3}{4}x^4 - C\\ \frac{1}{y} \amp= -\frac{3}{4}x^4 + C\\ y \amp= \frac{-1}{\frac{3}{4}x^4 + C} \end{align*}
    Note: in the third line, \(-C\) was replaced with \(C\text{.}\) Since \(C\) ranges over all real numbers, so does \(-C\) — so this is just a relabeling. See the remark below.
    🔗
    🔗
    Step 5: Write the general solution.
    🔗
    Combining the equilibrium solution from Step 1 and the explicit solution from Step 4, the general solution is:
    \begin{equation*} y = \frac{-1}{\frac{3}{4}x^4 + C} \quad \text{and} \quad y = 0. \end{equation*}
    🔗
    🔗
    🔗
    And we just successfully used separation of variables! All the problems with separation of variables follow this basic format (with steps 1-5). Some of the details will vary in each problem, which we explore in the following sections.
    🔗

    Remark 1.4.4. Absorbing Constants.

    In solving the previous example, we changed \(\frac{1}{y} = -\frac{3}{4}x^4 - C\) to \(\frac{1}{y} = -\frac{3}{4}x^4 + C\text{.}\) This is valid because \(-C\) and \(C\) both vary over all real numbers (they represent the same family of values, just relabeled).
    🔗
    We often absorb constants into \(C\) like this, and it is generally preferred to write the simpler expression. For example, it is much neater to write \(y = \sin(x) + C\) than \(y = \sin(x) + 2C\text{,}\) even though they represent the same family of functions. It is standard practice to reuse the name \(C\) from line to line, even when absorbing constants into it.
    🔗
    🔗

    Note 1.4.5. Be Careful with \(C\) Placement.

    Be careful when changing expressions with \(C\text{.}\) For example
    • \(y = \tan(x) + C\) and \(y = \tan(x + C)\) are not the same.
      🔗
    • \(y = 2 + Ce^x\) and \(y = Ce^x\) are not the same (the constant \(2\) cannot be absorbed into \(Ce^x\) due to the order of operations).
      🔗
    🔗
    🔗
    Let’s do a quick check to make sure you are confident simplifying expressions with \(C\text{.}\)
    🔗

    Checkpoint 1.4.6.

    In each of the following, two expressions involving \(C\) (where \(C\) varies over all real numbers) are given. Select all pairs that represent the same family of functions.
    🔗
    • \(\frac{1}{y} = -x^2 - C\) and \(\frac{1}{y} = -x^2 + C\)
    • These are equivalent! Since \(C\) varies over all real numbers, so does \(-C\) — they represent the same family of values, just relabeled.
    • \(y = e^x + C - 3\) and \(y = e^x + C\)
    • These are equivalent! The \(-3\) is just a fixed number that gets absorbed into \(C\text{.}\) As \(C\) varies over all reals, so does \(C - 3\text{.}\)
    • \(y = 1 + Ce^x\) and \(y = Ce^x\)
    • Not equivalent! The \(1\) cannot be absorbed into \(Ce^x\) — adding 1 to \(Ce^x\) shifts every solution up by 1, which is a genuinely different family of functions.
    • \(y = \cos(x) + 2C\) and \(y = \cos(x) + C\)
    • These are equivalent! As \(C\) varies over all real numbers, so does \(2C\) — it’s just a relabeling of the constant.
    • \(y = \sqrt{x} + C\) and \(y = \sqrt{x + C}\)
    • Not equivalent! In the first, \(C\) shifts the graph vertically. In the second, \(C\) shifts the graph horizontally inside the square root. These are genuinely different families of functions.
    🔗
