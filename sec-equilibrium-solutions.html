    Section 1.1.5 Equilibrium Solutions

    We now take a brief detour to discuss equilibrium solutions. This is a simple but important concept that will come up throughout the course.
    🔗

    Definition 1.1.1. Equilibrium Solution.

    An equilibrium solution to a differential equation is a constant solution. That is, \(y(t) = C\) for all \(t\text{,}\) where \(y(t)\) is a solution to the ODE.
    🔗
    🔗

    Note 1.1.2.

    A function \(y(t)\) is constant if and only if \(y'(t) = 0\) for all \(t\text{.}\) We will use this fact frequently when finding equilibrium solutions.
    🔗
    🔗
    As a quick example, consider the differential equation \(\frac{dP}{dt} = 2P\text{,}\) whose general solution is \(P(t) = Ce^{2t}\text{.}\) Use the slider below to explore how different values of \(C\) give different specific solutions. Can you find the value of \(C\) that makes \(P(t)\) a constant function?
    🔗
    Notice that these solutions are not constant in general. However, setting \(C = 0\) gives \(P(t) = 0\text{,}\) which is constant, and therefore the only equilibrium solution.
    🔗
    Understanding equilibrium solutions gives us useful insight into the overall behavior of solutions to an ODE. Later in the course we will explore this in more depth. For now, let’s see how equilibrium solutions arise naturally in the context of Newton’s law of cooling.
    🔗

    Example 1.1.4. Newton’s Law of Cooling.

    Recall that Newton’s law of cooling states that \(\frac{dT}{dt} = -k(T - A)\text{,}\) where \(T\) is the temperature of an object, \(A\) is the surrounding temperature, and \(k\) is a positive constant. Suppose the surrounding temperature is \(A = 72^{\circ}\)F, so the ODE becomes:
    \begin{equation*} \frac{dT}{dt} = -k(T - 72). \end{equation*}
    What is the equilibrium solution? Justify this both intuitively and algebraically.
    🔗

    (a) Intuitive Justification.

    What value of \(T\) ensures the object never changes temperature?
    🔗
    Hint.
    If the surrounding temperature is \(72^{\circ}\)F, what does the temperature of the object need to be so that it doesn’t change?
    🔗
    🔗
    Solution.
    An object stops changing temperature once it matches the temperature of its surroundings. If the surrounding temperature is \(72^{\circ}\)F and the temperature of the object, \(T\text{,}\) is \(72^{\circ}\)F, then the temperature of the object remains at \(72^{\circ}\)F. So \(T(t) = 72\) is the equilibrium solution.
    🔗
    🔗
    🔗

    (b) Algebraic Justification.

    Find the equilibrium solution algebraically. We haven’t done this yet, so I expect most will need the hint to start!
    🔗
    Hint.
    We want to find for what value of \(C\) the constant function \(T(t) = C\) is a solution. Notice that this implies \(\frac{dT}{dt} = 0\text{.}\)
    🔗
    This really is the same process as "verifying" a solution. But here, you sub in \(T(t) = C\) and \(\frac{dT}{dt} = 0\text{.}\) Then you solve for \(C\text{.}\)
    🔗
    Substitute both expressions into the ODE \(\frac{dT}{dt} = -k(T - 72)\) and solve for \(C\text{.}\)
    🔗
    🔗
    Solution.
    For a constant solution \(T(t) = C\text{,}\) we have \(\frac{dT}{dt} = 0\text{.}\) Substituting into the ODE:
    \begin{align*} \frac{dT}{dt} \amp= -k(T - 72)\\ 0 \amp= -k(C - 72)\\ 0 \amp= C - 72\\ C \amp= 72. \end{align*}
    Therefore \(T(t) = 72\) is the equilibrium solution.
    🔗
    🔗
    🔗
    🔗

    Checkpoint 1.1.5.

    Find equilibrium solutions to \(\frac{dy}{dt} = (y-2)^2\text{?}\)
    🔗
    Use the same method as the algebraic solution to the Newton’s law of Cooling problem above.
    🔗
    • \(y = 2\)
    • Correct! Setting \(y = C\) gives \(\frac{dy}{dt}=0\text{.}\) Plug into the ODE and solve:
      \begin{align*} \frac{dy}{dt} \amp= (y-2)^2\\ 0 \amp= (C - 2)^2\\ 0 \amp= C - 2\\ C \amp= 2. \end{align*}
    • \(y = 0\)
    • Not quite. Try setting \(y = C\) and \(\frac{dy}{dt} = 0\text{,}\) then substitute into the ODE and solve for \(C\text{.}\)
    • \(y = -2\)
    • Not quite. Try setting \(y = C\) and \(\frac{dy}{dt} = 0\text{,}\) then substitute into the ODE and solve for \(C\text{.}\)
    • There are no equilibrium solutions.
    • There is one! Try setting \(y = C\) and \(\frac{dy}{dt} = 0\text{,}\) then substitute into the ODE and solve for \(C\text{.}\)
    🔗
    Let’s now practice finding equilibrium solutions for more complicated differential equations. Notice that finding the general solution in the following examples would be quite difficult, but substituting \(y(t) = C\) and \(y' = y'' = \cdots = 0\) often reduces the problem to a simple algebraic equation in \(C\text{.}\)
    🔗

    Activity 1.1.1. Practice: Finding Equilibrium Solutions.

    Try each of the following on your own before checking the solution. Try it on your own first!
    🔗

    (a)

    Find all equilibrium solutions, \(v(t) = C\text{,}\) to the differential equation
    \begin{equation*} v'' + v' \cdot (1 + t^2 + v) + v^2 = 2v. \end{equation*}
    🔗
    Solution.
    Since we are looking for a constant solution, we set \(v(t) = C\text{,}\) which gives:
    \begin{equation*} v = C, \quad v' = 0, \quad v'' = 0. \end{equation*}
    Substituting into the differential equation:
    \begin{align*} 0 + 0 \cdot (1 + t^2 + C) + C^2 \amp= 2C\\ C^2 - 2C \amp= 0\\ C(C - 2) \amp= 0 \end{align*}
    So \(C = 0\) or \(C = 2\text{,}\) giving two equilibrium solutions:
    \begin{equation*} v(t) = 0 \quad \text{and} \quad v(t) = 2. \end{equation*}
    🔗
    🔗
    🔗

    (b)

    Find all equilibrium solutions to the differential equation
    \begin{equation*} v''t^2 + vv' + v = t^3. \end{equation*}
    🔗
    Solution.
    Setting \(v(t) = C\text{,}\) so \(v' = 0\) and \(v'' = 0\text{,}\) and substituting:
    \begin{equation*} 0 \cdot t^2 + C \cdot 0 + C = t^3 \implies C = t^3. \end{equation*}
    But \(C\) must be a constant, and \(t^3\) is not constant. Therefore there are no equilibrium solutions.
    🔗
    🔗
    🔗
    🔗

    Note 1.1.6.

    Equilibrium solutions do not always exist, as the last example shows. Recall that we are looking for a constant \(C\) such that \(y(t) = C\) satisfies the ODE. If the algebra forces \(C\) to equal a non-constant expression, then no equilibrium solution exists.
    🔗
    🔗
    Let’s drive this point home with a quick checkpoint!
    🔗

    Checkpoint 1.1.7.

    Consider the ODE \(\frac{dy}{dt} = -y + t - 1\text{.}\) Setting \(y = C\) and \(\frac{dy}{dt} = 0\) and substituting:
    \begin{align*} 0 \amp= -C + t - 1\\ C \amp= t - 1. \end{align*}
    What, if any, are the equilibrium solutions?
    🔗
    • \(y = t - 1\)
    • Close, but \(y = t - 1\) is not a constant function! An equilibrium solution must be constant for all values of \(t\text{.}\)
    • \(y = 0\)
    • Not quite. Try plugging \(y = 0\) into the ODE: \(0 = 0 + t - 1 = t - 1\text{,}\) which is not true for all \(t\text{.}\)
    • \(t = C + 1\)
    • This is a common mistake! We solved for \(t\) instead of \(C\text{.}\) We need \(C\) to be a constant — solving for \(t\) doesn’t give us an equilibrium solution.
    • There are no equilibrium solutions.
    • Correct! We need \(C\) to be a constant, but \(C = t - 1\) depends on \(t\text{.}\) So no constant function satisfies this ODE.
    🔗
