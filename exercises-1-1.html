    Exercises 1.1.6 Exercises for Section 1.1

    1.

    Write a differential equation where the solution \(y(x)\) has the property that the slope of its graph at any point \((x, y)\) equals the sum of \(x\) and \(y\text{.}\)
    🔗
    🔗

    2.

    Write a differential equation where the solution \(y(x)\) has the property that the tangent line to its graph at any point \((x, y)\) passes through the point \((-y,\, x)\text{.}\)
    🔗
    Hint.
    You will need to use the fact that the slope of any line is \(\frac{y_1-y_0}{x_1-x_0}\text{.}\)
    🔗
    Compute the slope of the tangent line: one of the points it passes through is \((-y,\, x)\text{,}\) what other point does the tangent line always pass through? Refer to the setting up ODEs example in the notes.
    🔗
    🔗
    🔗

    3.

    The temperature \(T\) of a cup of coffee changes at a rate proportional to the difference between the ambient air temperature and the coffee’s current temperature. Suppose the ambient air temperature is \(65^\circ\)F. Write a differential equation modeling \(T(t)\text{.}\)
    🔗
    🔗

    4.

    (a)

    Verify that \(y(x) = Ce^{-2x}\) is a solution of \(y' + 2y = 0\text{.}\)
    🔗
    🔗

    (b)

    Find the value of \(C\) satisfying the initial condition \(y(0) = 4\text{.}\)
    🔗
    🔗

    5.

    Verify by substitution that \(y_1 = \cos(5x)\) and \(y_2 = \sin(5x)\) are both solutions of:
    \begin{equation*} y'' + 25y = 0. \end{equation*}
    🔗
    🔗

    6.

    Verify by substitution that \(y = 10e^x - 6e^{-x}\) is a solution of:
    \begin{equation*} y' = y + 12e^{-x}. \end{equation*}
    🔗
    🔗

    7.

    Find all integer values of \(n\) for which \(y(x) = x^n\) is a solution to:
    \begin{equation*} xy'' - y' - \frac{3y}{x} = 0. \end{equation*}
    🔗
    🔗

    8.

    Find all integer values of \(n\) for which \(f(t) = t^n\) is a solution to:
    \begin{equation*} \frac{d^2f}{dt^2} + \frac{1}{t}\frac{df}{dt} = 0. \end{equation*}
    🔗
    🔗

    9.

    Find all values of the constant \(r\) for which \(y = e^{rx}\) is a solution of:
    \begin{equation*} y'' - y' - 6y = 0. \end{equation*}
    🔗
    🔗

    10.

    Find all equilibrium (constant) solutions to the equation below. That is, find all values of \(C\) for which \(y(t) = C\) is a solution.
    \begin{equation*} y'' = e^{\cos t}\, y' - \sqrt{2t}\, y' + y^2 - t^2. \end{equation*}
    🔗
    🔗

    11.

    Find all equilibrium (constant) solutions to the equation below. That is, find all values of \(C\) for which \(\phi(q) = C\) is a solution.
    \begin{equation*} \frac{d\phi}{dq} + 9 = \phi^2 - q\frac{d\phi}{dq}. \end{equation*}

    The following calculus facts will be useful throughout these exercises. Feel free to use them as a quick reference!
    🔗
    Calculus Prerequisites
    🔗
    • Power Rule: \(\dfrac{d}{dx}(x^n) = nx^{n-1}\)
      🔗
      Solution. Example
      \(\dfrac{d}{dx}(x^3) = 3x^2\text{.}\) We multiply by the exponent and reduce the power by 1.
      🔗
      🔗
      🔗
    • Exponential Rule: \(\dfrac{d}{dx}(e^x) = e^x\text{,}\) and more generally by the chain rule: \(\dfrac{d}{dx}(e^{kx}) = ke^{kx}\)
      🔗
      Solution. Example
      \(\dfrac{d}{dx}(e^{3x}) = 3e^{3x}\text{.}\) The chain rule brings down the inner derivative \(3\) as a factor.
      🔗
      🔗
      🔗
    • Chain Rule: \(\dfrac{d}{dx}(f(g(x))) = f'(g(x)) \cdot g'(x)\)
      🔗
      Solution. Example
      \(\dfrac{d}{dx}(e^{x^2}) = e^{x^2} \cdot 2x = 2xe^{x^2}\text{.}\) Here \(f(u) = e^u\) and \(g(x) = x^2\text{,}\) so \(f'(g(x)) = e^{x^2}\) and \(g'(x) = 2x\text{.}\)
      🔗
      🔗
      🔗
    • Product Rule: \(\dfrac{d}{dx}(f(x)g(x)) = f'(x)g(x) + f(x)g'(x)\)
      🔗
      Solution. Example
      \(\dfrac{d}{dx}(xe^x) = 1 \cdot e^x + x \cdot e^x = e^x(1+x)\text{.}\)
      🔗
      🔗
      🔗
    • Trig Derivatives: \(\dfrac{d}{dx}(\sin x) = \cos x\) and \(\dfrac{d}{dx}(\cos x) = -\sin x\)
      🔗
      🔗
