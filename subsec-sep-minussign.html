    Section 1.4.4 Separation of Variables: Some Subtle Points

    Now we look at two more examples solving separable ODEs. This first example illustrates a subtle point when dealing with square roots. See if you can find the mistake in the work below!
    🔗

    Exploration 1.4.1. A Subtle Moment in Solving an IVP.

    Let’s solve the following initial value problem explicitly: \(x\frac{dx}{dt} = t\text{,}\) \(x(0) = -2\text{.}\)
    🔗
    After separating variables and integrating, we arrive at:
    \begin{equation*} x^2 = t^2 + C. \end{equation*}
    🔗
    Solution. Work for separation of variables
    First, why is this separable? Notice that \(x\) is already on the left with \(dx\text{,}\) so no dividing by \(f(x)\) is necessary — we can separate immediately by writing \(x\, dx = t\, dt\text{.}\) There are no equilibrium solutions since setting \(x = C\) gives \(0 = t\text{,}\) which can’t hold for all \(t\text{.}\)
    🔗
    Separating variables:
    \begin{equation*} x\, dx = t\, dt. \end{equation*}
    Integrating both sides:
    \begin{align*} \int x\, dx \amp= \int t\, dt\\ \frac{1}{2}x^2 \amp= \frac{1}{2}t^2 + C\\ x^2 \amp= t^2 + C. \end{align*}
    🔗
    🔗
    Now something interesting happens when we solve for \(x\) explicitly. Let’s take the square root of both sides:
    \begin{equation*} x(t) = \sqrt{t^2 + C}. \end{equation*}
    Applying the initial condition \(x(0) = -2\text{:}\)
    \begin{equation*} -2 = \sqrt{0 + C} = \sqrt{C}. \end{equation*}
    But the square root of a number is always non-negative — so \(\sqrt{C} = -2\) is impossible! We seem to have reached a contradiction. What went wrong?
    🔗

    (a) What went wrong?

    Identify the error in the working above.
    🔗
    • We integrated incorrectly. The antiderivative of \(x\) is not \(\frac{1}{2}x^2\text{.}\)
    • Not quite — \(\int x\, dx = \frac{1}{2}x^2 + C\) is correct!
    • We forgot the \(\pm\) when taking the square root.
    • Exactly! When solving \(x^2 = t^2 + C\text{,}\) we should have written \(x = \pm\sqrt{t^2 + C}\text{.}\) The negative branch is what allows \(x(0) = -2\text{.}\)
    • The initial condition \(x(0) = -2\) is inconsistent with this ODE, so no solution exists.
    • Not quite — a solution does exist! The issue is earlier in the working.
    🔗

    (b)

    Now that you’ve identified the mistake, see if you can finish solving the problem correctly! Check with the solution below.
    🔗
    Solution. Corrected solution
    The fix is to include the \(\pm\) when taking the square root:
    \begin{equation*} x = \pm\sqrt{t^2 + C}. \end{equation*}
    Now applying \(x(0) = -2\text{:}\) since \(x(0)\) is negative, we choose the negative branch:
    \begin{equation*} -2 = -\sqrt{C} \implies \sqrt{C} = 2 \implies C = 4. \end{equation*}
    So the specific solution is:
    \begin{equation*} x(t) = -\sqrt{t^2 + 4}. \end{equation*}
    🔗
    🔗
    🔗
    🔗
    So far, when solving a separable ODE we have always ended by solving explicitly for the dependent variable, arriving at something like \(y = Ce^{2x}\) or \(x = -\sqrt{t^2 + 4}\text{.}\) But sometimes it is difficult or even impossible to isolate the dependent variable, and that’s okay! An equation relating \(y\) and \(x\) that implicitly defines \(y\) as a function of \(x\text{,}\) without explicitly writing \(y = \ldots\text{,}\) is called an implicit solution. For example, \(y^2 + x^2 = C\) could be an implicit solution to an ODE even though we haven’t written \(y\) by itself. In these cases, we simply stop once we have a clean implicit relation and call that the general solution.
    🔗

    Example 1.4.1. Finding a General Implicit Solution.

    Find a general implicit solution to \(v'(t) = \frac{1}{v} - \frac{t^2}{v}\text{.}\) Before diving in, see if you can figure out whether there are any equilibrium solutions!
    🔗

    (a) Separating the variables.

    We are trying to solve \(v'(t) = \frac{1}{v} - \frac{t^2}{v}\text{.}\) Which of the following correctly separates the variables?
    🔗
    • \(\displaystyle\int v\, dv = \int (1 - t^2)\, dt\)
    • Correct! Multiplying both sides by \(v\) gives \(v\frac{dv}{dt} = 1 - t^2\text{,}\) and then separating gives \(v\, dv = (1-t^2)\, dt\text{.}\)
    • \(\displaystyle\int \frac{1}{v}\, dv = \int (1 - t^2)\, dt\)
    • Not quite — check your separation step. You need to multiply both sides by \(v\) first, not divide.
    • \(\displaystyle\int v\, dv = \int \left(-\frac{t^2}{v}\right) dt\)
    • Not quite — the right-hand side still contains \(v\text{,}\) which means the variables are not fully separated!
    • This ODE is not separable.
    • It is separable! Try rewriting the right-hand side by factoring out \(\frac{1}{v}\text{:}\) \(v' = \frac{1-t^2}{v}\text{.}\) Now can you see how to separate?
    Solution. How to separate the variables
    First notice \(v' \) is the same thing as \(\frac{dv}{dt} \text{,}\) and we can rewrite the right-hand side by combining the fractions:
    \begin{equation*} \frac{dv}{dt} = \frac{1}{v} - \frac{t^2}{v} = \frac{1 - t^2}{v}. \end{equation*}
    Now multiply both sides by \(v\text{:}\)
    \begin{equation*} v \cdot \frac{dv}{dt} = 1 - t^2 \implies v\, dv = (1 - t^2)\, dt. \end{equation*}
    Integrating both sides:
    \begin{equation*} \int v\, dv = \int (1 - t^2)\, dt. \end{equation*}
    🔗
    🔗
    🔗

    (b) Integrating and writing the implicit solution.

    After integrating both sides, we arrive at:
    \begin{equation*} \frac{1}{2}v^2 = t - \frac{1}{3}t^3 + C. \end{equation*}
    This is the general implicit solution. See the dropdown for all the details.
    🔗
    Solution. Full working
    Equilibrium solutions: None.
    🔗
    Solution. Why are there no equilibrium solutions?
    Rewriting the ODE as \(\frac{dv}{dt} = \frac{1}{v}(1 - t^2)\text{,}\) we have \(f(v) = \frac{1}{v}\text{.}\) Setting \(f(v) = 0\) gives \(\frac{1}{v} = 0\text{,}\) which has no solution. So there are no equilibrium solutions.
    🔗
    🔗
    Separating and integrating:
    \begin{align*} v\, dv \amp= (1 - t^2)\, dt\\ \int v\, dv \amp= \int (1 - t^2)\, dt\\ \frac{1}{2}v^2 \amp= t - \frac{1}{3}t^3 + C. \end{align*}
    Since solving explicitly for \(v\) would require taking a square root (introducing a \(\pm\)) and the problem only asks for an implicit solution, we stop here. The general implicit solution is:
    \begin{equation*} \frac{1}{2}v^2 = t - \frac{1}{3}t^3 + C. \end{equation*}
